# Editor Mk7 Autosave

一个单文件的 Markdown 编辑器，重点放在“写着写着别丢稿”。

## 自动保存能力

- 智能节流保存：停止输入 2 秒后自动保存
- 定时保存：每 30 秒自动保存一次
- 多版本管理：最多保存 3 个版本
- 草稿恢复：24 小时内自动恢复未保存的草稿
- 状态指示：右上角显示自动保存状态
- 错误处理：存储空间不足时自动清理旧数据

## 其他特性

- 图片保留原始格式
- 支持撤销 / 重做
- 双栏编辑 / 预览切换
- 浏览器直接打开即可使用

## 使用方式

1. 直接用浏览器打开 [editor-mk7-autosave.html](./editor-mk7-autosave.html)
2. 开始编辑，或上传 `.md` 文件
3. 右上角会显示自动保存状态
4. 需要导出时点击“另存为 .md”

## 文件结构

- [editor-mk7-autosave.html](./editor-mk7-autosave.html): 主应用，包含样式、交互和自动保存逻辑
